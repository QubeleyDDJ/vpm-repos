【簡単.anim保存】Face2Anim 【Unity Editor拡張】- 改変した表情をそのままアニメーション書き出し
================================================================================

Unity上のアバターの「今の表情（シェイプキーの状態）」を、ワンクリックでアニメーションファイル（.anim）として保存できるUnity Editor拡張ツールです。

通常、Unityで表情アニメーションを作るには「Animationウィンドウを開く」「録画ボタンを押す」「大量のシェイプキーから項目を探して数値を打つ」という手間がかかります。
Face2Animを使えば、それらすべての工程をスキップして一発出力。
そのままベースファイルとしてFaceEmoでの合成等に使用できます。


こんな時に便利です
------------------
【表情の固定・合成に】
改変した「理想のデフォルト顔」を書き出し、FaceEmo等のツールで「ベース表情」として合成。
全てのハンドサインにあなたのこだわりを反映させたい時に。

【VRChatの表情メニュー作成に】
標準スライダーで作った表情をそのまま .anim 化。
Expression Menu に登録する素材作りが劇的にスピードアップします。

【ポーズ・撮影用に】
「この瞬間の表情を保存しておきたい！」という時に。
いつでもお気に入りの顔を再現できるようになります。


使い方（3ステップ）
------------------
1. Tools → Face2Anim を選択
2. アニメーションファイルにしたい SkinnedMeshRenderer をドラッグ＆ドロップ
3. 保存先を指定して保存


インストール
------------
1. ダウンロードした ZIP（.unitypackage が入った配布ファイル）を解凍します
2. 解凍して出てきた `.unitypackage` をダブルクリックするか、Unity メニュー
   `Assets > Import Package > Custom Package…` から `.unitypackage` を指定してインポートします
3. インポートダイアログで「Import」を押し、コンパイル完了を待ちます
4. Unity メニュー `Tools > Face2Anim` から起動できます

含まれるもの:
  - Editor/Face2AnimExporter.cs … 書き出しロジック
  - Editor/Face2AnimWindow.cs   … メニュー Tools/Face2Anim から開くウィンドウ

補足:
  - 一時ファイルは書き出し時に `Assets/Face2Anim/` 配下へ短時間だけ作成され、完了後に自動削除されます。
  - `.anim` の保存先は任意のフォルダを選べます（Assets 外も可）。


動作環境
--------
- Unity version 2022.322f1
- VRCSDK3 Avatars
- Humanoidアバター
- FaceEmoなどの外部ツールとの併用を推奨します
